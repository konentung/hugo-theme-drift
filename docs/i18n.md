# i18n (Internationalization)

hugo-theme-drift supports multiple languages out of the box. All UI strings (navigation, labels, messages) are translatable.

## Supported Languages

| Language | Code | File |
|----------|------|------|
| English | `en` | `i18n/en.toml` |
| 繁體中文 | `zh-TW` | `i18n/zh-TW.toml` |
| 简体中文 | `zh-CN` | `i18n/zh-CN.toml` |
| 日本語 | `ja` | `i18n/ja.toml` |
| 한국어 | `ko` | `i18n/ko.toml` |
| Deutsch | `de` | `i18n/de.toml` |
| Français | `fr` | `i18n/fr.toml` |
| Español | `es` | `i18n/es.toml` |
| Português | `pt` | `i18n/pt.toml` |
| Русский | `ru` | `i18n/ru.toml` |
| Italiano | `it` | `i18n/it.toml` |
| Türkçe | `tr` | `i18n/tr.toml` |

## Configuration

Set your language in `hugo.toml`:

```toml
languageCode = 'zh-tw'
defaultContentLanguage = 'zh-tw'
```

Hugo will automatically load the matching `i18n/*.toml` file.

## Adding a New Language

1. Copy an existing file (e.g., `i18n/en.toml`)
2. Rename it to your locale code (e.g., `i18n/ko.toml`)
3. Translate all string values
4. Set `languageCode` and `defaultContentLanguage` in your `hugo.toml`

## Translated Strings

The following UI strings are translated:

- **Navigation**: Home, Posts, Projects, Notes, Tags, About, Search
- **Hero**: Explore Posts, View Projects
- **Section headers**: Recent Posts, Featured Projects, Notes, View all, Latest
- **Post metadata**: min read, Previous, Next
- **Table of Contents** heading
- **Share & Comments** labels
- **Search**: placeholder, tips
- **404 page**: title, message, back button
- **Footer**: All rights reserved
- **Alerts**: Note, Tip, Important, Warning, Caution

## Contributing Translations

Pull requests for new languages are welcome! Follow the format in `i18n/en.toml`.
