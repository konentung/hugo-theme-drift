# Share Buttons

Social sharing buttons can be displayed at the bottom of each post.

## Enable

In `hugo.toml`:

```toml
[params]
  showShareButtons = true
```

## Supported Platforms

- **X (Twitter)** — Share with post title and URL
- **Facebook** — Share URL
- **WhatsApp** — Send post title and URL

## Appearance

Share buttons appear as circular icons above the post tags in the footer area. They use the theme's glassmorphism style and each platform has its branded hover color.
