# hugo-theme-drift

[![GitHub Release](https://img.shields.io/github/v/release/konen/hugo-theme-drift?style=flat-square)](https://github.com/konentung/hugo-theme-drift/releases)
[![License](https://img.shields.io/github/license/konen/hugo-theme-drift?style=flat-square)](https://github.com/konentung/hugo-theme-drift/blob/main/LICENSE)

A modern Hugo theme with 3D floating sidebar navigation, dark/light mode, and dynamic animations.

<!-- ![Screenshot](images/screenshot.png) -->

## Features

- 3D Floating Sidebar with glassmorphism
- Dark / Light mode toggle with persistence
- Client-side full-text search
- GitHub integration for project pages
- 4 Content types — Posts, Projects, Notes, About
- Typewriter effect, scroll reveal, 3D card tilt
- Responsive mobile drawer
- Zero dependencies — pure vanilla JS

## Documentation

👉 **[hugo-theme-drift docs](https://konentung.github.io/hugo-theme-drift/)**

## Quick Start

```bash
hugo new site mysite
cd mysite
git init
git submodule add https://github.com/konentung/hugo-theme-drift.git themes/hugo-theme-drift
cp themes/hugo-theme-drift/hugo.example.toml hugo.toml
hugo server
```

See the [documentation](https://konentung.github.io/hugo-theme-drift/) for full configuration details.
